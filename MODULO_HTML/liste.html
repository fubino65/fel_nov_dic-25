LE LISTE

Liste non ordinate (tag ul + tag li per i singoli elementi)

I figli diretti di una lista non ordinata possono essere solo li

  • elemento 1
  • elemento 2
  • elemento 3
  • elemento 4
  • elemento 5

Liste ordinate (tag ol + tag li per i singoli elementi)

I figli diretti di una lista ordinata possono essere solo li

  1. argomento 1
  2. argomento 2
  3. argomento 3

Liste di definizione (tag dl + dt(definition term) e dd(definition description))

N.B. dt e dd sono entrambi figli diretti di dl, quindi fratelli fra di loro

HTML
Hypertext Markup Language
Il linguaggio standardd per la creazione di pagine web
CSS
Cascading Style Sheets
Linguaggio utilizzato per definire la formattazione e lo stile delle pagine html
JAVASCRIPT
Linguaggio di programmazione frontend per aggiungere interattività alle pagine web
